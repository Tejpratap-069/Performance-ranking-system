Performance Ranking System

A web-based Performance Ranking System designed to evaluate and compare the performance of students or employees using analytical metrics.
The system automatically calculates scores, ranks individuals, and displays results through a clean interactive dashboard.

This project was developed using HTML, CSS, and JavaScript and runs entirely in the browser using LocalStorage for data management.

Features

User Login and Registration system

School Mode for managing student performance

Company Mode for managing employee performance

Automatic ranking system

Performance score calculation

Top performers dashboard

Search and sort functionality

Add, edit, and delete records

Grade evaluation for students

Promotion suggestions for employees

Data saved using browser LocalStorage

System Overview

The system works in three main stages:

User Authentication
Users create an account and login to access the system.

Mode Selection
Users choose between:

School Mode

Company Mode

Performance Analysis
The system calculates performance scores and generates rankings automatically.
